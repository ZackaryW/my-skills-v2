# Hello World Skill

## Greeting with Time of Day
Time-based prefixes: Morning (5-11 AM), Afternoon (12-5 PM), Evening (6-8 PM), Night (9 PM-4 AM).

Uses system local time.

## Custom Message

### Usage Examples
```python
python script.py Charlie --message "Welcome"  # Welcome, Charlie!
python script.py Dr. Smith --message "Greetings"
```

Names are auto-capitalized, exclamation mark auto-appended.
